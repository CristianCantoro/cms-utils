# README

Script to start and stop `cms` services

start_judge.sh
--------------

Usage:
    ./start_judge <contest_id>

it can be used as follows:

```
$HOME/start_judge.sh $(cat $HOME/contest_id)
```

where `$HOME/contest_id` contains the identifier of the contest to start.

kill_cms.sh
-----------

Usage:
    ./kill_cms.sh
